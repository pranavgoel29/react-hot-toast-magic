# Getting Started

Add beautiful notifications to your React app with [react-hot-toast-magic](https://github.com/pranavgoel29/react-hot-toast-magic).

### Install with Yarn

```sh
yarn add react-hot-toast-magic
```

### Install with NPM

``` sh
npm install react-hot-toast-magic
```

## Basic usage
``` jsx
import toast, { Toaster } from 'react-hot-toast-magic';

const notify = () => toast('Here is your toast.');

const App = () => {
  return (
    <div>
      <button onClick={notify}>Make me a toast</button>
      <Toaster />
    </div>
  );
};
```
